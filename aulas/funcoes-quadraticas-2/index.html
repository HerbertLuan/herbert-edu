VÉRTICE

Coordenadas do vértice

Coordenada x

xv = −b / 2a

É também o eixo de simetria.

Coordenada y

yv = −Δ / 4a

Atalho: ou calcule f(xv).

Dica: ache xv primeiro; depois substitua em f(x) para o yv sem decorar a fórmula do Δ.

MÁXIMO / MÍNIMO

O vértice é o valor extremo

a > 0 → vértice é o ponto de MÍNIMO

a < 0 → vértice é o ponto de MÁXIMO

No ENEM: “lucro máximo”, “altura máxima”, “custo mínimo” = y do vértice.

SIMETRIA

Eixo de simetria

Pontos com mesma altura ficam à mesma distância do eixo xv.

x = −b / 2a

Reta vertical que passa pelo vértice e espelha a parábola.

Atalho de raízes: se você conhece as duas raízes, o vértice está exatamente no meio delas: xv = (x₁+x₂)/2.

RAÍZES

Fórmula de Bhaskara

x = ( −b ± √Δ ) / 2a

com o discriminante

Δ = b² − 4ac

1
Identifique a, b, c.
2
Calcule Δ = b² − 4ac.
3
Aplique Bhaskara com o ± → duas raízes.
4
Confira: substitua uma raiz e veja se dá 0.
Δ E O GRÁFICO

O discriminante prevê as raízes

Δ > 0

2 raízes reais — cruza o eixo x em 2 pontos.

Δ = 0

1 raiz — apenas toca o eixo x (no vértice).

Δ < 0

Sem raiz real — não toca o eixo x.

EXEMPLO RESOLVIDO

f(x) = x² − 6x + 8

1
Δ = (−6)² − 4·1·8 = 36 − 32 = 4 → 2 raízes.
2
x = (6 ± √4)/2 = (6 ± 2)/2 → x = 4 ou x = 2.
3
xv = −(−6)/2 = 3 (meio entre 2 e 4 ✓).
4
yv = f(3) = 9 − 18 + 8 = −1 → mínimo.

Raízes (2,0) e (4,0); vértice (3, −1); corte y em (0, 8).

PROBLEMA · ESTILO ENEM

Lucro de uma empresa

O lucro (em mil reais) por vender x unidades é

L(x) = −x² + 40x − 300

Quantas unidades geram o lucro máximo? E qual é esse lucro?

Reflexão: por que faz sentido o lucro ter um teto e cair depois?

1
a = −1 < 0 → existe máximo (vértice).
2
xv = −40/(2·−1) = 20 unidades.
3
L(20) = −400 + 800 − 300 = R$ 100 mil.

Máximo: 20 unidades → R$ 100 mil.

ESTRATÉGIA

Resolver rápido no ENEM

1. Leia a pergunta antes da função

“Máximo/mínimo” → vértice. “Quando vale 0 / zero” → raízes.

2. Use o sinal de a primeiro

Elimina alternativas de cara (abre p/ cima ou baixo).

3. Vértice no meio das raízes

xv = (x₁+x₂)/2 evita refazer −b/2a.

4. c é leitura direta

Corte em y = (0, c), sem conta.
