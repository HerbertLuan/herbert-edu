Pergunta para começar: por que tantos problemas do ENEM — lucro, altura de um projétil, área máxima — caem sempre na mesma curva?

CONCEITO

O que é uma função quadrática

Definição

É toda função que pode ser escrita como

f(x) = ax² + bx + c

com a, b, c reais e a condição essencial a ≠ 0.

Por que a ≠ 0?

Se a = 0, o termo ax² some e sobra bx + c — uma função do 1º grau (reta), não uma parábola.

O ax² é a alma da função: é ele que curva o gráfico.

f(x)=x²−4x+3 ✓ f(x)=−2x²+5 ✓ f(x)=3x+1 ✗ (1º grau)
COEFICIENTES

O que cada letra controla

a — abertura

Sinal: concavidade.
a>0 abre para cima ∪
a<0 abre para baixo ∩

|a| grande → parábola estreita.

b — inclinação

Desloca o vértice na horizontal e define a inclinação onde o gráfico cruza o eixo y.

Atua junto com a na posição do eixo de simetria.

c — ponto de partida

É o valor de f(0): onde a parábola corta o eixo y.

O gráfico passa por (0, c), sem conta.

Dica de leitura rápida: olhe a (para onde abre) e c (onde toca o eixo y) antes de calcular qualquer coisa.

GRÁFICO

A parábola e suas partes

f(x) = 0,6x² − 2,4x + 1 (a > 0, concavidade para cima)

  • Vértice: ponto mais baixo (ou mais alto) da curva.
  • Eixo de simetria: reta vertical que “dobra” a parábola ao meio.
  • Raízes: onde a curva cruza o eixo x.
  • Interseção y: ponto (0, c).

Toda parábola é simétrica. Isso vai economizar conta na Aula 2.

CONCAVIDADE

O sinal de a decide tudo

a > 0 → abre para cima ∪ (tem mínimo)

a < 0 → abre para baixo ∩ (tem máximo)

Mnemônico: a positivo = sorriso ☺ · a negativo = tristeza ☹

INTERSEÇÕES

Onde a parábola cruza os eixos

Eixo Y — sempre 1 ponto

Faça x = 0:

f(0) = c

O gráfico corta o eixo y em (0, c). Direto.

Eixo X — 0, 1 ou 2 pontos

Faça f(x) = 0 e resolva ax²+bx+c = 0 (as raízes).

Quantas raízes? Depende do discriminante — assunto da Aula 2.

Δ > 0 → 2 raízes Δ = 0 → 1 raiz Δ < 0 → nenhuma raiz real
VARIAÇÃO

Crescimento e decrescimento

A mudança de comportamento acontece exatamente no vértice.

a > 0 (∪)

Decresce até o vértice → depois cresce.

a < 0 (∩)

Cresce até o vértice → depois decresce.

O x do vértice é a fronteira entre subir e descer.

EXEMPLO · ESTILO ENEM

Lançamento de um projétil

A altura (em metros) de uma bola após t segundos é dada por

h(t) = −5t² + 20t

Identifique, sem resolver equação ainda, o que cada parte diz.

Pergunta reflexiva: o que o sinal de a = −5 garante sobre a trajetória?

1
a = −5 < 0 → concavidade para baixo: a bola sobe e desce (altura máxima).
2
c = 0 → em t=0 a altura é 0: a bola parte do chão.
3
As raízes (h=0) dirão quando ela sai e quando volta ao chão.

Leitura antes da conta: metade da questão já resolvida.

ATENÇÃO

Erros mais comuns

✗ Concavidade ao contrário

a > 0 abre para cima — não para baixo.

✗ Trocar o c

O corte no eixo y é (0, c), não (c, 0).

✗ Toda parábola tem raízes

Se Δ < 0, ela não cruza o eixo x.

✗ Esquecer a ≠ 0

Sem o termo x², não é função quadrática.
